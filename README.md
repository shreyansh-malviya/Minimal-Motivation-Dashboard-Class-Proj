# Minimal Daily Motivation Dashboard

A class project built with **React + Vite**.

Displays a random motivational quote fetched from the [DummyJSON Quotes API](https://dummyjson.com/quotes/random). Includes a like system to save and review favourite quotes.

## Features

- Fetches a random quote on load
- **New Quote** button to get another quote
- **Like / Unlike** to save quotes to a personal list
- Liked quotes displayed below with an option to remove them
- Falls back to local quotes if the API is unavailable

## Run locally

```bash
npm install
npm run dev
```
